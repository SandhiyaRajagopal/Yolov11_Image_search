# Yolov11_Image_search

## ABSTRACT:
The YOLOv11 Object Search Application is a Computer Vision-based web application developed using YOLOv11 and Streamlit. The system detects objects from uploaded images, generates metadata, and enables object-based image search with bounding boxes and confidence scores.

## ENVIRONMENT SETUP:
## Prerequisites
1. Python 3.11+
2. Visual Studio Code
3. Streamlit
4. YOLOv11
5. Internet Connection

## FEATURES:
1. Real-time object detection
2. Object-based image search
3. Bounding box visualization
4. Confidence score display
5. Streamlit web interface

## TECHNOLOGIES USED:
1. Python
2. YOLOv11
3. Streamlit
4. OpenCV
5. Ultralytics

## INSTALLATION:

git clone https://github.com/SandhiyaRajagopal/Yolov11_Image_search.git
cd Yolov11_Image_search
pip install -r requirements.txt

## RUN THE PROJECT:
streamlit run app.py
